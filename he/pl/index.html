Keamanan & Teknis

Dibangun dengan Standar Profesional

Hanya User Login

Tools generator hanya bisa diakses oleh user WordPress yang login (Admin, Editor, Author, tim redaksi).

API Key Aman

Request Gemini diproses melalui server WordPress. API key tidak terekspos di frontend JavaScript.

Nonce WordPress

Request AJAX dilindungi nonce untuk mencegah request tidak sah dari luar dashboard.

Proteksi Domain

Plugin hanya berjalan di domain terdaftar. Lisensi via GitHub dengan dukungan wildcard domain.

Update Otomatis

Notifikasi update di WordPress, one-click update dari GitHub. Metadata dari update.json.

Cache Lisensi

Daftar lisensi disimpan dalam cache sementara agar tidak membebani server dengan request berulang.

// Sistem lisensi dari GitHub URL: https://raw.githubusercontent.com/yanuarzg/cc/main/he/pl/licenses.json // Format: { "domains": [ "harianexpress.com", "*.harianexpress.com", "clientdomain.com", "*.clientdomain.com" ] }
