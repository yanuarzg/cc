Akurasi & Keamanan Konten

Source Lock + Safe Paraphrase

Dua fitur kunci untuk menjaga fakta tetap akurat sekaligus menghasilkan konten yang unik.

Source Lock untuk Link Artikel

Plugin mengambil teks asli dari URL sumber dan mengunci data penting agar AI tidak mengarang. Tanggal, angka, nama narasumber, dan kutipan langsung akan dipertahankan persis seperti sumber.

Safe Paraphrase

Fakta dijaga, kutipan tetap verbatim, tetapi narasi di luar kutipan disusun ulang dengan struktur kalimat baru. Hasilnya: artikel orisinal yang aman dari duplikasi, tanpa kehilangan informasi penting.

Multi AI & Keamanan API

Dukungan Gemini, Grok/xAI, Z.AI

Fleksibel memilih provider AI, dengan sistem rotasi dan auto-block API key yang aman.

Dropdown Pilih AI

Di halaman generator, tersedia dropdown Pilih AI: Gemini, Grok/xAI, Z.AI. Bisa berganti provider kapan saja.

Rotasi & Retry Otomatis

Beberapa API key per provider (satu per baris). Jika satu key limit/high demand/timeout, plugin coba key lain dan retry.

Auto-block Key Bermasalah

Key yang bocor, invalid, expired, atau forbidden otomatis ditandai dan dilewati pada request berikutnya.

Masking & Tidak Ada Default Key

API key ditampilkan tersensor. Tidak ada key bawaan di ZIP, semuanya diinput manual via Pengaturan API.

Network Subdomain

Directory A-Z & Monitoring Kanal

Pantau seluruh subdomain media Anda dalam satu halaman, lengkap dengan filter, sortir, dan status update warna.

Directory A-Z Subdomain

Daftar subdomain dari DOMAIN_GROUPS.all, dengan search, filter grup, sortir A-Z, terbaru, artikel terbanyak/tersedikit.

Pintasan Cepat

Tombol buka website, login (/pinarak.php), dashboard WP, dan tools AI Journalist langsung ke subdomain terkait.

Artikel Terbaru & Total Artikel

Menampilkan judul terbaru, total artikel, waktu publish (time ago), dan kategori dari REST API.

Status Warna & Tombol +Domain

Warna hijau/kuning/merah untuk update <1 minggu / 1-4 minggu / >1 bulan. Tombol request domain via WhatsApp, link copyright YZG.
