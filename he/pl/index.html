Teknis & Keamanan

Siap untuk kebutuhan publisher WordPress.

Dibangun dengan PHP, JavaScript, HTML, CSS, integrasi Gemini API, AJAX WordPress, nonce, sistem auto update, dan pembatasan lisensi domain.

Rotasi beberapa Gemini API Key saat limit, timeout, atau server sibuk.
Request AI diproses melalui server WordPress, bukan expose API key mentah di frontend.
Self-hosted update checker via GitHub raw update.json.
Domain whitelist via GitHub licenses.json dengan dukungan wildcard.
Minimum rekomendasi: WordPress 6.0+, PHP 7.4+, REST API aktif, Admin AJAX aktif.
Cocok untuk administrator, editor, author, dan tim redaksi yang sudah login.
